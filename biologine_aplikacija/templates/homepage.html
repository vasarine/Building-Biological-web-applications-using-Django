HMM Analysis Tools

Biological web application for biological sequence analysis using hidden Markov models (HMMs).

Available Tools

HMMBUILD

Build a HMM profile from a multiple sequence alignment file (MSA). Useful for modeling conserved sequence patterns and insertion/deletion events.

Use Tool

HMMEMIT

Generate random sequences from a HMM profile. Useful for model validation and sequence simulation based on the created HMM model.

Use Tool

HMMSEARCH

Search sequences against a HMM profile. Useful for identifying homologous sequences and domain-level matches.
